# Aplicativo de Gestão de Ordens de Serviço

A ideia é desenvolver um aplicativo, fuincional, para gerenciar ordens de serviços. Neste app temos 4 modelos que representam as entidades que serão manipuladas nos processos do app:

- Modelo Cliente
- Modelo Técnico
- Modelo Comentários
- Modelo Ordens de Serviço

# Tecnologias utilizadas 🧰
- <a href="https://www.python.org/" target="_blank">Python 3.13+</a>  
- <a href="https://www.djangoproject.com/" target="_blank">Django 6</a>  
- <a href="https://www.django-rest-framework.org/" target="_blank">Django Rest Framework</a> 
- <a href="https://python-poetry.org/" target="_blank">Poetry</a> - Gerenciamento de dependências
